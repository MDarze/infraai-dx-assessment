# 6.2 — Aggregate dashboard (backoffice)

**Priority:** P2 · **Effort:** M · **Theme:** Backoffice

## Problem
The backend has all the answers across all assessments. That's a goldmine for content marketing ("70% of KSA mid-size construction firms still use WhatsApp for daily reporting") and for product strategy. Currently invisible.

## Solution
Backoffice dashboard with:
- KPI tiles: total assessments, average score, most common pain signal, most-recommended quick win
- Histograms: score distribution, axis-score distributions
- Lists: top quick wins by frequency, top AI opportunities by frequency
- Trends: assessments per month, average score over time

All read-only, exportable as images for blog posts / decks.

## Acceptance criteria
- [ ] Dashboard route in backoffice
- [ ] Tiles + 2–3 charts
- [ ] All data anonymized at the dashboard level
- [ ] Exportable as PNG (chart screenshot) for marketing use

## Files
- Backend: aggregation queries + endpoint
- Backoffice frontend: dashboard page

## Out of scope
- Drill-down from chart bar to source assessments (later)
- Saved dashboards / custom views
