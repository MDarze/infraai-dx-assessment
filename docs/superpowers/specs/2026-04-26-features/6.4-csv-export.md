# 6.4 — CSV / Excel export (backoffice)

**Priority:** P2 · **Effort:** S · **Theme:** Backoffice

## Problem
Buyers and your own team want to drop assessment data into Excel / Google Sheets for ad-hoc analysis. Currently the only export is JSON (frontend) or PDF (frontend); both are useless for spreadsheet workflows.

## Solution
Backoffice button to export:
- Single assessment as CSV (one row per question/answer with metadata)
- Filtered list as CSV (one row per assessment with axis scores, ROI, key fields)

Generate server-side with a streaming CSV writer. Filename includes timestamp.

## Acceptance criteria
- [ ] Per-assessment CSV download
- [ ] Bulk CSV (filtered list) download
- [ ] Properly UTF-8 BOM-prefixed for Arabic in Excel
- [ ] Newline / comma escaping correct

## Files
- Backend: export endpoint
- Backoffice frontend: download trigger

## Out of scope
- Native .xlsx (use ExcelJS later if asked)
- Scheduled exports / email digests
