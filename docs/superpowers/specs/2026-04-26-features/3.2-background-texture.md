# 3.2 — Subtle background texture

**Priority:** P2 · **Effort:** S · **Theme:** Branding

## Problem
Flat cream/charcoal backgrounds feel like a Bootstrap admin theme. The infraai-dx.com brand has texture (paper grain, geometric subtlety) that makes pages feel deliberate, not generic.

## Solution
Add a CSS-only background pattern to `body`:
- Light theme: very faint grid or noise on cream (`bg-bg`)
- Dark theme: faint scanline or noise on charcoal

Implementation: a single SVG data URL or `background-image` with a tiled pattern at very low opacity (~2–4%). Must not interfere with text legibility. No image files; keep it inline.

Or alternative: a `radial-gradient` from `bg` to a slightly-different shade at the corners, giving subtle vignette without being a "gradient page."

**Not** a gradient that darkens / colors the background — that would fight the palette.

## Acceptance criteria
- [ ] Pattern visible on close inspection but invisible at normal reading distance
- [ ] Identical pattern in both themes (recolored only)
- [ ] No measurable performance impact
- [ ] No data URL larger than 1KB
- [ ] Doesn't print (use `@media print { background: none }`)

## Files
- `src/styles.css` — body background-image

## Out of scope
- Per-page hero imagery (could come later, but not "background")
- Animated patterns
