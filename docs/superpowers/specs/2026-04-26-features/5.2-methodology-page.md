# 5.2 — Methodology page

**Priority:** P2 · **Effort:** S · **Theme:** Trust & conversion

## Problem
Sophisticated buyers will ask "how do you score these axes? what's the DNA classification based on?" Without a documented methodology, the report can be dismissed as opinion.

## Solution
A static `/methodology` page (or section in About) explaining:
- The 8 axes and what each measures
- How questions map to axes (high-level)
- The 1–5 scoring scale
- How aggregate score is computed (avg of non-zero axes × 20)
- DNA classification thresholds (e.g., "Decision DNA = Reactive when Decision axis < 3")
- ROI assumptions and limitations

Link from Results page header and footer. Also link from the About / Settings.
