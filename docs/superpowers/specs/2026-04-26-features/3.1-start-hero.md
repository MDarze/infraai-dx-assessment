# 3.1 — Hero treatment on the Start page

**Priority:** P1 · **Effort:** S · **Theme:** Branding

## Problem
The Start page currently jumps straight into form fields. There's no emotional moment, no brand statement, no "you're about to do something valuable." The infraai-dx.com aesthetic uses generous whitespace and a bold opening; the assessment tool should mirror that mood at the entry point.

## Solution
Above the existing form cards, add a hero block:
- Large headline (text-3xl or larger): `t("start.hero.title")` — e.g., "أين منشأتك في رحلة التحول الرقمي؟" / "Where is your firm on the digital transformation journey?"
- Sub-line in `text-lg text-ink-muted`: 2 sentences max, what the assessment delivers (score, DNA, ROI, recommendations)
- Time estimate: `~10 minutes · 25 questions · {N} respondents recommended`
- Generous vertical whitespace (mt-12 mb-16) so it breathes

Form sections appear below, unchanged in content but visually subordinate.

## Acceptance criteria
- [ ] Hero renders above form on Start
- [ ] Headline scales appropriately on mobile (text-2xl) and desktop (text-3xl/4xl)
- [ ] Time/question count caption present
- [ ] Bilingual (uses t())
- [ ] Doesn't push the Begin button below the fold on common viewports

## Files
- `src/pages/Start.tsx` — add hero JSX
- `src/i18n/{ar,en}.ts` — `start.hero.title`, `start.hero.lede`, `start.hero.meta` keys

## Out of scope
- Background imagery (covered by 3.2)
- Animation / scroll effects
