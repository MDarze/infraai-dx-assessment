# 3.6 — Empty and loading states

**Priority:** P2 · **Effort:** S · **Theme:** Branding

## Problem
Several places either show nothing or show technical messages while data loads or is missing:
- Results during sync `pending` shows just "Saving…" with no skeleton structure
- AI opportunities empty shows "AI opportunities not enabled (readiness/data may be insufficient)" — accurate but cold
- ROI empty shows "Enter numeric values to get an estimated ROI." — also dry
- New users on Start see a fully populated form (no onboarding hint)

## Solution
For each empty state:
- Friendly copy that gives the user a next action
- Optional small icon (Theme 3.4)
- For loading: skeleton boxes (animated `bg-surface-alt`) where the content will go

For Results sync pending: render skeleton tiles in the hero row instead of just a text line.
For AI opps empty: rephrase to "Build the foundations first → see what unlocks" with bullets pointing to required axis scores.
For ROI empty: "Want a SAR estimate? Add hourly cost in Settings or have your finance respondent answer D4."
