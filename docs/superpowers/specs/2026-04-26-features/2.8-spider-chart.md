# 2.8 — Spider/radar chart for axis scores

**Priority:** P1 · **Effort:** S · **Theme:** Results value

## Problem
The current axis scores grid is 8 stat tiles — informational but not memorable. Construction execs respond to visuals, especially for slide decks.

## Solution
Replace (or complement) the axis score grid with a 8-axis radar chart. Use a small chart library or hand-roll an SVG (~80 LOC). Brand color for the data polygon, faint gridlines, axis labels rotated to read clearly in both RTL and LTR.

Considerations:
- Library: `react-chartjs-2` (heavy, ~50KB), `recharts` (medium, ~30KB), or custom SVG (zero deps, ~3KB). **Recommend custom SVG** to keep bundle small and exact brand control.
- The radar should be square, ~320×320 on desktop, scaled down on mobile.
- Inside the polygon, optionally label the area with the aggregate score.

## Acceptance criteria
- [ ] Radar chart renders with 8 axes labeled
- [ ] Data polygon shaded `bg-brand/30`, outlined `border-brand`
- [ ] Axis labels are locale-aware (use `axisLabel(k)`)
- [ ] Renders correctly in RTL and LTR
- [ ] Print-friendly (visible in PDF export)

- `src/pages/Results.tsx` — also include in PDF export HTML
