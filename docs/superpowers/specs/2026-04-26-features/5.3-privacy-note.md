# 5.3 — Privacy / data handling note on Start

**Priority:** P1 · **Effort:** S · **Theme:** Trust & conversion

## Problem
Construction firms in KSA are increasingly cautious about data handling — what's collected, where it's stored, who sees it, retention. The current Start page asks for client name, contact details, financial info with no statement addressing any of this. That kills sophisticated buyers.

## Solution
Below the form fields, before the Begin button, a small block:

> **Your data:** Stored on Railway servers in [region]. Visible only to you and [Linkdev / InfraAI]. Not shared with third parties or used to train models. [Read full policy]

Bilingual. The "Read full policy" links to a Privacy page (or the marketing site's privacy policy if one exists).

## Acceptance criteria
- [ ] Privacy block visible on Start, above Begin button
- [ ] Concrete statements (not boilerplate)
- [ ] Link to full policy
- [ ] Bilingual
