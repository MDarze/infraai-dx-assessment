# 5.1 — Schedule follow-up CTA on Results

**Priority:** P1 · **Effort:** S · **Theme:** Trust & conversion

## Problem
After completing an assessment and seeing the results, the user has the highest possible intent to engage further — but there's no next-step CTA. They close the tab and the moment is gone. Cold-emailing them later converts at a fraction of the rate.

## Solution
Add a prominent CTA card on the Results page footer area: "Discuss your results with our team — book a 30-minute call." Embed a Calendly inline widget (or Google Calendar appointment scheduling) so they can book without leaving the page.

Pre-fill assessor's email and assessment ID into the booking metadata so the meeting prep is automated.

## Acceptance criteria
- [ ] CTA card visible on Results, prominently above the secondary actions
- [ ] Calendly widget embedded inline (not a popup)
- [ ] Pre-fills email + assessment ID into booking
- [ ] Bilingual CTA copy
- [ ] Falls back gracefully if Calendly fails to load

## Files
- `src/pages/Results.tsx` — CTA card + iframe / widget script
- `src/i18n/{ar,en}.ts` — CTA copy
- Calendly account setup (external)

## Out of scope
- In-house scheduling system
- Multi-team round-robin (Calendly handles that natively)
