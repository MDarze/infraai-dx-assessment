# 2.7 — "After we engage" projection

**Priority:** P2 · **Effort:** M · **Theme:** Results value

## Problem
The Results show the *current* state — diagnosis without prognosis. Sales motion needs the "after" picture: "if you implement these top 3 recommendations, your projected score is 81/100 and your weekly saving doubles."

## Solution
For each quickWin, attach a projected delta per axis (e.g., the "standard daily report" quickWin → `+0.8 to Data axis, +0.3 to DailyOps`). Sum projected deltas across recommended quickWins, recompute the aggregate score, and show side-by-side: Today | Projected.

Apply only to the user-selected quickWins (checkboxes, default all checked). User can deselect to see partial impact.

This is more sales tool than analytical truth. Be conservative with deltas. Add an "estimated, not guaranteed" caveat.

## Acceptance criteria
- [ ] Projected score tile next to current score on Results hero
- [ ] Quick wins have checkbox; recompute on toggle
- [ ] Caveat copy visible
- [ ] PDF export shows both Today and Projected

## Files
- `src/analysis.ts` — add `axisDelta?: Partial<AxisScores>` to each quickWin definition
- `src/pages/Results.tsx` — projection calculation + side-by-side hero
- New i18n keys for Today/Projected and caveat

## Out of scope
- Time-to-impact ("this delta in 60 days") — keep deltas as static
- AI opp deltas (smaller, more speculative — defer)
